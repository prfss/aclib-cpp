Detailed Description

template<typename T, typename compare = std::less<T>>
class aclext::MonotonicConvexHullTrick< T, compare >

追加クエリおよび最小値/最大値クエリが単調な場合のConvex Hull Trickを実装したクラスです

一次関数の集合 \(L = \{f_1,f_2, \dots ,f_n\}\) に対して, 次の2つの処理を行うことができます.

  • 関数 \(a_i x + b_i\) の追加
  • 与えられた \(x\) に対し, 全ての \(1 \le j \le n\) について, \(\mathrm{compare}(f_i(x),f_j(x)) \lor f_i(x) = f_j(x)\) を満たす \(f_i(x)\) の取得

後者について特に compare が std::less の場合は最小値を,std::greater の場合は最大値を得ることができます.

制約

  • operator() で与えられるクエリ \(x_1,x_2, \dots ,x_m\) は単調
  • add で追加される直線の傾きが順に \(a_1,a_2, \dots ,a_n\) であるとき, 全ての \(1 \le i \lt n\) に対して
    • \(x_i\) が狭義単調増加する場合

      \[\begin{aligned} \begin{cases} a_i \le a_{i+1} & (\mathrm{compare} = \mathrm{std::greater}) \\ a_i \ge a_{i+1} & (\mathrm{compare} = \mathrm{std::less}) \end{cases} \end{aligned}\]

    • \(x_i\) が狭義単調減少する場合

      \[\begin{aligned} \begin{cases} a_i \ge a_{i+1} & (\mathrm{compare} = \mathrm{std::greater}) \\ a_i \le a_{i+1} & (\mathrm{compare} = \mathrm{std::less}) \end{cases} \end{aligned}\]
