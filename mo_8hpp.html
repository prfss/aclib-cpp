区間クエリのリスト \(\mathrm{query} = \{[l_1,r_1], [l_2,r_2], \dots ,[l_q, r_q]\}\) に対し, 答えのリスト \(\{\mathit{ans}(l_1,r_1),\mathit{ans}(l_2,r_2), \dots ,\mathit{ans}(l_q,r_q)\}\) を返します. ただし \(n = \max_{{i \in [1,q]}} r_i\) とするとき

  • \(\mathrm{extend}(\mathrm{init},0) = ans(0,0)\)
  • \(0 \lt l \le r \le n\) のとき, \(\mathrm{extend}(\mathit{ans}(l,r),l-1) = \mathit{ans}(l-1,r)\)
  • \(0 \le l \lt r \le n\) のとき, \(\mathrm{shrink}(\mathit{ans}(l,r),l) = \mathit{ans}(l+1,r)\)
  • \(0 \le l \le r \lt n\) のとき, \(\mathrm{extend}(\mathit{ans}(l,r),r+1) = \mathit{ans}(l,r+1)\)
  • \(0 \le l \lt r \le n\) のとき, \(\mathrm{shrink}(\mathit{ans}(l,r),r) = \mathit{ans}(l,r-1)\)
    を満たす必要があります.

制約

  • \(0 \le l_i \le r_i\)
