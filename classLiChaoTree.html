Detailed Description

template<typename X, typename Y, typename F, typename compare = std::less<Y>>
class LiChaoTree< X, Y, F, compare >

Li-Chao Treeを実装したクラスです

関数の集合 \(L = \{f_1,f_2, \dots ,f_n\}\)に対して, 次の2つの処理を行うことができます.

  • 関数の追加
  • ある \(x\)と全ての \(1 \le j \le n\)に対し, \(\mathrm{compare}(f_i(x),f_j(x)) \lor f_i(x) = f_j(x)\)を満たす \(f_i(x)\)の取得

後者について特にcompareが std::lessの場合は最小値を, std::greaterの場合は最大値を得ることができます.

制約

  • Xは整数型
  • Fは Y operator()(X)を持つ
  • addで追加される相異なる関数 \(f\)と \(g\)について, \(f(x) = g(x)\)となるような \(x\)は高々一つ
