Detailed Description

template<typename T, T(*)(T, T) add, T(*)(T, T) mul, T zero, T one, bool has_subtraction = false>
class Matrix< T, add, mul, zero, one, has_subtraction >

和・差・積・累乗などの演算を備えた行列クラスです

Tは要素の型です.add,zeroは加算とその単位元,mul,oneは乗算とその単位元を表します. またhas_subtraction == trueの時,operator-を定義します.

制約

  • \(\mathrm{add}(x,\text{zero}()) = \mathrm{add}(\text{zero}(),x) = x\)
  • \(\mathrm{mul}(x,\text{one}()) = \mathrm{mul}(\text{one}(),x) = x\)
