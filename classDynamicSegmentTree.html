Detailed Description

template<typename S, S(*)(S, S) op, S(*)() e, typename I = int>
class DynamicSegmentTree< S, op, e, I >

動的セグメント木を実装したクラスです

T型の配列 \(a\)とモノイド \((\mathrm{T}, \mathrm{op}, \mathrm{e})\)に対して,

  • 要素の変更( \(a_i = \mathrm{val}\))
  • 区間 \([l, r)\)の総積 \(\mathrm{op}(a[l], ..., a[r-1])\)の取得

を行うことができるデータ構造です.
